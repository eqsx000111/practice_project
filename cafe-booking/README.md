# ☕ Cafe Booking — система бронирования столиков

## 📌 Описание
API-сервис для бронирования столиков в кафе.

Позволяет создавать, редактировать и удалять бронирования, а также управлять доступностью столиков.

---

## 🚀 Функциональность

- Создание бронирования  
- Проверка доступности столиков  
- Обновление и удаление брони  
- Валидация входных данных  
- Работа с базой данных  

---

## 🛠️ Стек технологий

- Python  
- FastAPI  
- SQLAlchemy  
- PostgreSQL / SQLite  
- Pydantic  
- Alembic  

---

## ⚙️ Запуск проекта
## Клонирование репозитория
```bash
git clone https://github.com/eqsx000111/cafe-booking.git
cd cafe-booking
```

## Создание виртуального окружения
```bash
python -m venv venv
source venv/bin/activate  # Linux / Mac
venv\Scripts\activate     # Windows
```
## Установка зависимостей
```bash
pip install -r requirements.txt
```
## Запуск сервера
```bash
uvicorn app.main:app --reload
```

# 🌐 PEP Parser

## 📌 Описание
Скрипт для парсинга PEP (Python Enhancement Proposals) с официального сайта Python.

Собирает информацию о статусах PEP и формирует отчет.

---

## 🚀 Функциональность

- Сбор данных с сайта python.org  
- Парсинг HTML с помощью BeautifulSoup  
- Формирование отчетов  
- Сохранение данных в файл  

---

## 🛠️ Стек технологий

- Python  
- requests  
- BeautifulSoup4  

---

## ⚙️ Запуск проекта

```bash
git clone https://github.com/eqsx000111/bs4_parser_pep-master.git
cd bs4_parser_pep-master
```
# Установка зависимостей и запуск
```bash
pip install -r requirements.txt
python main.py
```
# Cправка по командам
```bash
python main.py -h
python main.py --help
```

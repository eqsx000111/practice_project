# 🤖 Homework Bot

## 📌 Описание
Telegram-бот для отслеживания статуса проверки домашних заданий через API.

Бот периодически отправляет уведомления об изменении статуса работы.

---

## 🚀 Функциональность

- Получение статуса задания через API  
- Отправка уведомлений в Telegram  
- Логирование ошибок  
- Обработка исключений  

---

## 🛠️ Стек технологий

- Python  
- python-telegram-bot  
- requests  
- logging  

---

## ⚙️ Запуск проекта

```bash
git clone https://github.com/eqsx000111/homework-bot.git
cd homework-bot
```
Создание .env:
```bash
TELEGRAM_TOKEN=your_token
API_TOKEN=your_api_token
```
Установка зависимостей и запуск бота:
```bash
pip install -r requirements.txt
python main.py
```
